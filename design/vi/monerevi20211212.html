# Monere视觉识别Logo设计

# 前言

Vee前端团队秉承完善大前端体系建设相关理念，为了追求线上应用稳定及产品信息收集而打造了一款包含数据统计(埋点)、性能统计、异常统计的前端监控系统Monere，本视觉识别系统仅为Monere所做Logo定义及继承Vee主色所做子视觉延展系列。

# 标志

Monere标志为Vee大前端团队旗下所做前端监控系统所使用，该logo仅为Vee团队及组织使用。标志设计以字母"M"为基本形状，其为"Monere"所示之意，在拉丁语中，Monere意为"提醒、警告"之意，此为表征数据信息的提示及监听之意义；整体以一笔构型，将数据与应用进行有机的结合，充分体现监控体系的作用与重要含义，镂空处以类似"眼睛"的形状出现，更加体现出"监控、监视"的寓意所在。色彩上与主色调保持一致，同样采用明亮高雅的金色与热情洋溢的红色相呼应，渐变的过渡呈现欢快、热情、明亮的整体感受。
