# еүҚз«ҜйңҖиҰҒдәҶи§Јзҡ„RTMPзҹҘиҜҶ

# жҰӮиҝ°

RTMPжҳҜReal Time Messaging Protocolзҡ„з®Җз§°пјҢжҳҜеҹәдәҺдҪ•ж—¶зҡ„дј иҫ“еұӮеҚҸи®®дёҠ(еҰӮtcpеҚҸи®®)зҡ„еә”з”ЁеұӮзә§еҲ«зҡ„еҚҸи®®пјҢе…¶и®ҫи®ЎжҳҜдёәдәҶз»„еҢ…еӨҡеӘ’дҪ“дј иҫ“жөҒ(еҰӮи§Ҷйў‘пјҢйҹійў‘д»ҘеҸҠдәӨдә’еҶ…е®№)

# д»Ӣз»Қ

RTMPеҚҸи®®жҳҜз”ұAdobeе…¬еҸёжҸҗеҮәзҡ„дёҖз§Қеә”з”ЁеұӮзҡ„еҚҸи®®пјҢдёҠйқўеј•иҝ°зҡ„жҳҜRTMPе®ҳж–№ж–ҮжЎЈзҡ„introductionйғЁеҲҶпјҢе…¶зӣ®зҡ„жҳҜдёәдәҶз»ҷиҜёеҰӮи§Ҷйў‘гҖҒйҹійў‘д»ҘеҸҠж•°жҚ®ж¶ҲжҒҜйҖҡдҝЎеҸҢж–№жҸҗдҫӣе®һж—¶зҡ„дҝЎжҒҜеҸҢеҗ‘жөҒгҖӮRTMPеңЁе®һзҺ°дёҠеҜ№дёҚеҗҢзұ»еһӢзҡ„ж¶ҲжҒҜе®һж–ҪдёҚеҗҢзҡ„жқғйҮҚпјҢиҝҷж ·еҸҜд»Ҙе®һзҺ°еҜ№еҪ“дј иҫ“йҮҸеҸ—йҷҗж—¶зҡ„еә•еұӮжөҒдј иҫ“ж¶ҲжҒҜиҝӣиЎҢжҺ’йҳҹгҖӮ

# е®ҡд№ү

  • иҪҪиҚ·(Payload)пјҡеҢ…еҗ«еңЁеҢ…дёӯзҡ„ж•°жҚ®

  • еҢ…(Packet)пјҡж•°жҚ®еҢ…з”ұеӣәе®ҡеӨҙе’ҢиҪҪиҚ·ж•°жҚ®з»„жҲҗпјҢдёҖдәӣеә•еұӮеҚҸи®®еҸҜиғҪйңҖиҰҒеҜ№иў«е®ҡд№үеҢ…иҝӣиЎҢе°ҒиЈ…

  • з«ҜеҸЈ(Port)пјҡTCP/IPз«ҜеҸЈпјҢз”ұдё»жңәеҸ·жқҘеҢәеҲҶеӨҡең°еқҖ

  • дј иҫ“ең°еқҖ(Transport address)пјҡзҪ‘з»ңең°еқҖе’Ңз«ҜеҸЈзҡ„з»„еҗҲ

  • ж¶ҲжҒҜжөҒ(Message stream)пјҡж¶ҲжҒҜжөҒйҖҡдҝЎзҡ„йҖ»иҫ‘йў‘йҒ“

  • ж¶ҲжҒҜжөҒID(Message stream ID)пјҡжҜҸдёӘж¶ҲжҒҜйғҪжңүдёҖдёӘйў„жңҹж¶ҲжҒҜжөҒеҢ№й…Қзҡ„id

  • еқ—(Chunk)пјҡж¶ҲжҒҜзүҮж®өгҖӮж¶ҲжҒҜеңЁиў«зҪ‘з»ңдј иҫ“еүҚдјҡиў«еҲҶжҲҗе°Ҹзҡ„йғЁеҲҶпјҢжүҖжңүзҡ„ж¶ҲжҒҜдјҡйҖҡиҝҮеӨҡи·Ҝз”ұж—¶й—ҙеәҸз«ҜеҜ№з«Ҝдј иҫ“

  • еқ—жөҒ(Chunk stream)пјҡйҖҡдҝЎзҡ„йҖ»иҫ‘йў‘йҒ“е…Ғи®ёеқ—д»ҘжҹҗдёҖзү№е®ҡж–№еҗ‘дј йҖҒпјҢеқ—жөҒеҸҜд»Ҙд»Һе®ўжҲ·з«Ҝдј еҲ°жңҚеҠЎз«ҜпјҢд№ҹеҸҜд»Ҙд»ҺжңҚеҠЎз«Ҝдј еӣһе®ўжҲ·з«Ҝ

  • еқ—жөҒID(Chunk stream ID)пјҡжҜҸдёӘеқ—жөҒйғҪжңүдёҖдёӘдёҺд№ӢеҢ№й…Қзҡ„id

  • еӨҡи·ҜеӨҚз”Ё(Multiplexing)пјҡеҲҶзҰ»зҡ„йҹіи§Ҷйў‘ж•°жҚ®еҸҜд»Ҙиҝһз»ӯж’ӯж”ҫзҡ„иҝҮзЁӢпјҢдҪҝеҫ—йҹіи§Ҷйў‘еҸҜд»ҘеҗҢж—¶дј йҖҒ

  • еӨҡи·ҜеҲҶз”Ё(DeMultiplexing)пјҡеӨҡи·ҜеӨҚз”Ёзҡ„йҖҶиҝҮзЁӢпјҢеҸҜд»Ҙе°ҶдәӨй”ҷзҡ„йҹіи§Ҷйў‘ж•°жҚ®иҒҡйӣҶжҲҗеҺҹе§Ӣзҡ„йҹіи§Ҷйў‘ж•°жҚ®

  • иҝңзЁӢиҝҮзЁӢи°ғз”Ё(Remote Procedure Call)пјҡе…Ғи®ёе®ўжҲ·з«ҜжҲ–жңҚеҠЎз«ҜеҸҜд»Ҙи°ғз”ЁеҸҰдёҖз«ҜжңҚеҠЎз«ҜжҲ–е®ўжҲ·з«ҜеӯҗзЁӢеәҸжҲ–еӯҗиҝӣзЁӢзҡ„иҜ·жұӮ

  • е…ғж•°жҚ®(Metadata)пјҡж•°жҚ®зҡ„жҸҸиҝ°гҖӮз”өеҪұзҡ„е…ғж•°жҚ®еҢ…жӢ¬з”өеҪұж ҮйўҳпјҢж—¶й•ҝпјҢдёҠжҳ ж—ҘжңҹзӯүзӯүгҖӮ

  • еә”з”Ёе®һдҫӢ(Application Instance)пјҡе®ўжҲ·з«ҜеҸҜд»ҘиҝһжҺҘе№¶еҸ‘йҖҒиҝһжҺҘиҜ·жұӮзҡ„жңҚеҠЎз«Ҝе®һдҫӢ

  • еҠЁдҪңж¶ҲжҒҜж јејҸ(Action Message Format)пјҡз”ЁжқҘеәҸеҲ—еҢ–AS(ActionScript еҠЁдҪңи„ҡжң¬)е®һдҫӢеҜ№иұЎ(object graphs)зҡ„еҺӢзј©зҡ„дәҢиҝӣеҲ¶ж јејҸпјҢжңүAMF0е’ҢAMF3дёӨз§ҚзүҲжң¬

# еӯ—иҠӮеәҸ & еҜ№йҪҗ & ж—¶й—ҙж јејҸ

жүҖжңүзҡ„ж•ҙеӯ—ж®өд»Ҙеӯ—иҠӮеәҸжқҘдј иҫ“пјҢе№¶дё”жҳҜеӨ§з«ҜжҺ’еҲ—пјҢдёҚеҒҡзү№ж®ҠиҜҙжҳҺеқҮдёәеҚҒиҝӣеҲ¶гҖӮж—¶й—ҙжҲід»ҘжҜ«з§’и®°пјҢд»Һ0ејҖе§ӢпјҢ32дҪҚй•ҝпјӣж—¶й—ҙй—ҙйҡ”з”ЁжҜ«з§’зҡ„ж— з¬ҰеҸ·ж•ҙж•°иЎЁзӨәпјҢеҸҜиғҪжҳҜ24жҲ–32дҪҚй•ҝеәҰ

# RTMPеқ—жөҒ

е°Ҫз®ЎRTMPжҳҜиў«и®ҫи®Ўз”ЁжқҘе®һж—¶дј иҫ“ж¶ҲжҒҜзҡ„пјҢдҪҶе®ғд№ҹеҸҜд»Ҙз”ЁжқҘеӨ„зҗҶд»»дҪ•ж¶ҲжҒҜгҖӮRTMPйқһеёёйҖӮеҗҲз”ЁдәҺеӨ§йҮҸзҡ„йҹіи§Ҷйў‘еә”з”ЁпјҢиҷҪз„¶е№¶дёҚдјҡжҸҗдҫӣд»»дҪ•дјҳеҢ–пјҢдҪҶеҚҙиў«з”ЁдәҺжӣҙй«ҳзә§еҲ«зҡ„еҚҸи®®жқҘиҝӣиЎҢдјҳеҢ–пјҢдҫӢеҰӮзӣҙж’ӯи§Ҷйў‘жңҚеҠЎеҷЁеҸҜиғҪдјҡйҖүжӢ©еңЁе®ўжҲ·з«ҜзҪ‘йҖҹиҫғж…ўзҡ„жғ…еҶөдёӢдёўејғи§Ҷйў‘ж¶ҲжҒҜиҖҢдҝқиҜҒйҹійў‘ж¶ҲжҒҜзҡ„е®һж—¶еҲ°иҫҫгҖӮ

# ж¶ҲжҒҜж јејҸ

ж¶ҲжҒҜж јејҸеҸҜиў«еҲҶжҲҗи®ёеӨҡеқ—жқҘжҸҗдҫӣеӨҡи·ҜеӨҚз”ЁпјҢиҝҷдәӣж¶ҲжҒҜйңҖиҰҒеҢ…жӢ¬еҗ«д»ҘдёӢеӯ—ж®өпјҡ

еӯ—ж®өеҗҚ еҗ«д№ү еӨ§е°Ҹ
ж—¶й—ҙжҲі(Timestamp) ж¶ҲжҒҜзҡ„ж—¶й—ҙжҲі 4еӯ—иҠӮ
й•ҝеәҰ(Length) ж¶ҲжҒҜиҪҪиҚ·зҡ„й•ҝеәҰпјҢеҰӮжһңж¶ҲжҒҜеӨҙдёҚиғҪиў«зңҒз•ҘпјҢйңҖиҰҒеҢ…еҗ«й•ҝеәҰ 3еӯ—иҠӮ
зұ»еһӢID(Type ID) зұ»еһӢidиў«з”ЁдәҺж¶ҲжҒҜжҺ§еҲ¶ 1еӯ—иҠӮ
ж¶ҲжҒҜжөҒID(Message Stream ID) д»»ж„Ҹж•°еҖјпјҢе°Ҹз«ҜеӯҳеӮЁ 4еӯ—иҠӮ

# жҸЎжүӢ

RTMPжҸЎжүӢдёҚеҗҢдәҺе…¶д»–еҚҸи®®пјҢе®ғеҢ…еҗ«3дёӘеӣәе®ҡеӨ§е°Ҹзҡ„еқ—иҖҢдёҚжҳҜеҸҜеҸҳеӨ§е°Ҹзҡ„еқ—

# жҸЎжүӢж¬ЎеәҸ

ејҖе§Ӣж—¶пјҢе®ўжҲ·з«ҜеҸ‘йҖҒC0е’ҢC1

е®ўжҲ·з«Ҝеҝ…йЎ»еңЁж”¶еҲ°S1жүҚиғҪеҸ‘йҖҒC2пјҢж”¶еҲ°S2жүҚиғҪеҸ‘йҖҒд»»дҪ•е…¶д»–ж•°жҚ®

жңҚеҠЎз«Ҝеҝ…йЎ»зӯүеҲ°ж”¶еҲ°C0жүҚиғҪеҸ‘йҖҒS0е’ҢS1пјҢе№¶дё”жңҖеҘҪиғҪеңЁж”¶еҲ°C1еҗҺгҖӮжңҚеҠЎз«ҜзӯүеҲ°ж”¶еҲ°C1жүҚиғҪеҸ‘йҖҒS2пјҢжңҚеҠЎз«Ҝеҝ…йЎ»зӯүеҲ°ж”¶еҲ°C2жүҚиғҪеҸ‘йҖҒе…¶д»–ж•°жҚ®

# C0 & S0 ж јејҸ

C0е’ҢS0жҳҜдёҖдёӘ8дҪҚз»„пјҢиў«и®ӨдёәжҳҜдёҖдёӘ8дҪҚж•ҙж•°ж®өгҖӮеҜ№C0иҖҢиЁҖпјҢиҝҷдёӘеӯ—ж®өиў«и®ӨдёәжҳҜе®ўжҲ·з«ҜRTMPиҜ·жұӮзҡ„зүҲжң¬пјӣеҜ№S0иҖҢиЁҖпјҢиҝҷдёӘеӯ—ж®өиў«и®ӨдёәжҳҜжңҚеҠЎз«ҜйҖүжӢ©зҡ„RTMPзүҲжң¬гҖӮй»ҳи®ӨжҳҜ3пјҢ0-2иў«еәҹејғпјҢ4-31иў«з”ЁдәҺжңӘжқҘзҡ„зүҲжң¬пјҢ32-255жІЎжңүиў«е…Ғи®ё

# жҸЎжүӢз®Җеӣҫ

  • жңӘеҲқе§ӢеҢ–йҳ¶ж®ө(Uninitialized)пјҡеҚҸи®®зүҲжң¬еңЁиҝҷдёӘйҳ¶ж®өеҸ‘йҖҒпјҢе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜйғҪжҳҜжңӘеҲқе§ӢеҢ–зҡ„гҖӮе®ўжҲ·з«ҜеңЁC0еҢ…дёӯеҸ‘йҖҒеҚҸи®®зүҲжң¬пјҢеҰӮжһңжңҚеҠЎз«Ҝж”ҜжҢҒиҝҷдёӘеҚҸи®®зүҲжң¬пјҢе…¶е°ҶдјҡеҜ№еә”еҸ‘йҖҒS0е’ҢS1пјӣеҰӮжһңжңҚеҠЎз«ҜдёҚж”ҜжҢҒпјҢжңҚеҠЎз«ҜдјҡйҮҮеҸ–еҗҲйҖӮзҡ„еҠЁдҪңгҖӮеңЁRTMPдёӯпјҢиҝҷдёӘеҠЁдҪңжҳҜдёӯжӯўиҝһжҺҘгҖӮ

  • зүҲжң¬еҸ‘йҖҒйҳ¶ж®ө(Version Sent)пјҡеңЁжңӘеҲқе§ӢеҢ–йҳ¶ж®өд№ӢеҗҺе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜйғҪеӨ„дәҺзүҲжң¬еҸ‘йҖҒйҳ¶ж®өгҖӮе®ўжҲ·з«ҜеӨ„дәҺзӯүеҫ…S1еҢ…пјҢжңҚеҠЎз«ҜеӨ„дәҺзӯүеҫ…C1еҢ…гҖӮеңЁж”¶еҲ°зӣёеә”зҡ„зӯүеҫ…зҡ„еҢ…еҗҺпјҢе®ўжҲ·з«ҜеҸ‘йҖҒC2иҖҢжңҚеҠЎз«ҜдјҡеҸ‘йҖҒS2пјҢз„¶еҗҺиҝӣе…ҘзЎ®и®ӨеҸ‘йҖҒйҳ¶ж®өгҖӮ

  • зЎ®и®ӨеҸ‘йҖҒйҳ¶ж®ө(Ack Sent)пјҡе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«Ҝе“Қеә”зҡ„зӯүеҫ…S2е’ҢC2

  • жҸЎжүӢз»“жқҹйҳ¶ж®ө(Handshake Done)пјҡе®ўжҲ·з«Ҝе’ҢжңҚеҠЎз«ҜдәӨжҚўж¶ҲжҒҜ
